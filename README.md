## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript repository enhanced with some additional features.

### Included Features
- Postgres Database and TypeORM Repository Pattern
- GraphQL CodeFirst Approach with multiple URLs
- REST APIs
- JWT Authentication
- Multi-tenancy with Postgres
- Kafka Integration
- Redis Integration
- Cron Jobs
- Jenkins Pipeline and Dockerfile
- Kubernetes Deployment


## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
